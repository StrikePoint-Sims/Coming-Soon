## 7. Membership Access Rules

### Practice

- Unlimited off-peak access
- No included peak hours
- Guest privileges included
- 7-day booking window
- 1 active reservation at a time

### Standard

- 8 peak hours per month
- Unlimited off-peak access
- Guest privileges included
- 10-day booking window
- 1 active reservation at a time

### Elite

- 16 peak hours per month
- Unlimited off-peak access
- Guest privileges included
- 14-day booking window
- 2 active reservations at a time

### Active Reservation Definition

An active reservation means a future reservation currently on the member's account. This does not mean the facility only allows one booking at a time across all members. There are 3 bays planned at launch, so multiple customers can have overlapping reservations in different bays.

## 20. Support Escalation

### Escalate by Text for Urgent Issues

Use text escalation for:

- safety concerns
- injury
- active access failure / lockout unresolved by AI
- customer trapped or unable to exit
- serious equipment/facility issue during active booking
- urgent security concern

### Escalate by Email for Non-Urgent Issues

Use email escalation for:

- billing disputes
- legal threats
- property damage follow-up
- press/media inquiries
- investor/franchise inquiries
- teaching professional partnerships
- unresolved non-urgent technical issues
- service-animal-related questions

### If Exact Contact Method Is Not Available to Customer

Use:

> I'll pass this along to the right person so they can review it directly.

Do not expose owner's personal phone/email unless intentionally configured.

## 21. Lockout / Access Issue Flow

### First Response

> Sorry — let's get that fixed. First, confirm you're using the access code tied to your current reservation and that your booking time has started.

### If Still Not Working

If OpenPath/code regeneration tool exists:

- verify customer identity/booking through system
- regenerate or resend code if authorized
- confirm customer can access

If unresolved:

> I'm going to escalate this as an access issue right away.

Escalate by text.

### Do Not

- reveal access control configuration
- discuss backup keys
- discuss camera blind spots
- give security workarounds
- tell customer to force doors
